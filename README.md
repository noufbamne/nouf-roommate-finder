# 🏠 Roommate Finder
> Smart roommate matching platform for better living compatibility.

---

## 🚀 Overview
A modern web application designed to help users find compatible roommates through a structured and user-friendly interface.

This project focuses on improving the roommate search experience by combining usability with smart matching concepts.

---

## 🧠 Features
- 👥 Roommate profile exploration  
- 🔍 Matching-based interface  
- ⚡ Fast and responsive UI  
- 🎨 Clean, modern design  

---

## 🛠 Tech Stack
- React + Vite  
- TypeScript  
- Tailwind CSS  
- shadcn/ui  

---

## ▶️ Run Locally
```bash
npm install
npm run dev
```

---

## 🎯 Purpose
To explore how technology can simplify the process of finding compatible roommates.
